Project details

ScholarDAO is a simple decentralized scholarship platform built on Stellar blockchain. Students can be added as candidates, and community members vote to decide who receives a scholarship. Each wallet can vote only once, ensuring fairness and transparency. All voting data is stored on-chain, making the process open and tamper-proof. This project demonstrates how blockchain can replace traditional, biased selection systems with a transparent and community-driven approach.

Vision

ScholarDAO aims to make scholarship selection fair, transparent, and accessible to everyone. By using blockchain, it removes bias and central control, allowing communities to directly support deserving students. In the future, this model can scale globally, connecting donors, voters, and students in one decentralized system. It has the potential to transform how educational funding works, creating equal opportunities and empowering people worldwide.
